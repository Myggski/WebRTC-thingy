# WebRTC-thingy

I wanted to learn more about Vue 3, TypeScript, Node, Jest and WebRTC, so I created this project. 

## Project setup
```
cd /server
npm install

cd /app
npm install
```

### Compiles and hot-reloads for development
```
cd /app
npm run serve

cd /server
npm run dev
```

### Compiles and minifies for production
```
cd /app
npm run build

cd /server
npm run start
```
